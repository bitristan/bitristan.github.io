1.1 ART GC 的设计理念

Section titled “1.1 ART GC 的设计理念”

Android Runtime (ART) 的垃圾回收器设计遵循以下核心原则：

  • 低延迟优先：移动设备对交互响应敏感，GC 暂停时间直接影响用户体验
  • 内存效率：移动设备内存有限，需要高效利用每一字节
  • 并发执行：尽可能让 GC 与应用线程并行执行，减少暂停时间
  • 分代收集：利用”弱分代假说”——大多数对象朝生夕死，优化收集效率

2.2 各空间详解

Section titled “2.2 各空间详解”

2.2.1 Image Space (镜像空间)

Section titled “2.2.1 Image Space (镜像空间)”
  • 用途：存储预加载的类和对象，来自 boot.art 镜像文件
  • 特点：内存映射文件，只读，永不回收
  • 生命周期：从 Zygote 进程启动到系统关闭

2.2.2 Zygote Space (Zygote 空间)

Section titled “2.2.2 Zygote Space (Zygote 空间)”
  • 用途：Zygote fork 后共享的只读空间
  • 特点：在 Zygote fork 前创建，fork 后变为只读
  • 回收策略：仅 Full GC 时考虑回收

2.2.3 Malloc Space (RosAlloc/DlMalloc 空间)

Section titled “2.2.3 Malloc Space (RosAlloc/DlMalloc 空间)”
  • 用途：普通对象分配
  • 分配器：
    • RosAlloc (Runs-of-Slots Allocator)：线程本地分配，减少锁竞争
    • DlMalloc：Doug Lea’s malloc，传统分配器
  • 特点：支持非移动收集器 (MS/CMS)

2.2.4 Bump Pointer Space (指针碰撞空间)

Section titled “2.2.4 Bump Pointer Space (指针碰撞空间)”
  • 用途：Semi-Space 和 Mark-Compact 收集器使用
  • 特点：分配极快（只需移动指针），但需要整块连续内存
  • 适用场景：年轻代对象、移动式 GC

2.2.5 Region Space (区域空间)

Section titled “2.2.5 Region Space (区域空间)”
  • 用途：Concurrent Copying (CC) 收集器使用
  • 特点：将堆划分为多个固定大小的 Region，支持对象移动和压缩
  • 优势：减少内存碎片，支持并发整理

2.2.6 Large Object Space (大对象空间)

Section titled “2.2.6 Large Object Space (大对象空间)”
  • 用途：存储大对象（默认 > 12KB）
  • 特点：直接使用 mmap 分配，避免拷贝开销
  • 实现：LargeObjectMapSpace（离散空间）或 FreeListSpace

4.3 Semi-Space (SS)

Section titled “4.3 Semi-Space (SS)”

4.3.1 算法原理

Section titled “4.3.1 算法原理”

Semi-Space 将堆分为两个半空间：From-Space 和 To-Space

  1. 新对象分配在 To-Space
  2. GC 时，将存活对象从 From-Space 复制到 To-Space
  3. 交换两个空间的角色

4.3.2 优缺点

Section titled “4.3.2 优缺点”

优点：

  • 分配极快（指针碰撞）
  • 自动整理内存，无碎片
  • 适合年轻代对象（存活率低）

缺点：

  • 内存利用率低（只有一半可用）
  • 需要复制存活对象（存活率高时开销大）

7.1 Java 引用类型

Section titled “7.1 Java 引用类型”

ART 支持四种 Java 引用类型：

类型用途回收时机
StrongReference普通引用永不回收（直到不可达）
SoftReference软引用内存不足时回收
WeakReference弱引用下次 GC 时回收
PhantomReference虚引用随时可能回收，get() 永远返回 null

此外还有 FinalizerReference，用于实现 Object.finalize() 方法。

  • min_free / max_free：堆增长的最小/最大值
  • heap_growth_multiplier：前台堆增长倍数
