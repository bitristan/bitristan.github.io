ARM64 Linux 启动原理（八）：中断与 GIC

基于 Linux v6.10 源码，arch/arm64/kernel/entry.S、drivers/irqchip/irq-gic-v3.c

1. 概述

Section titled “1. 概述”

ARM64 的中断处理由两层组成：

  • 异常向量表（entry.S）：CPU 硬件在异常发生时自动跳转到向量表，保存/恢复寄存器，调用 C 处理函数
  • GICv3 中断控制器（irq-gic-v3.c）：管理中断的路由、优先级、分发，提供 IPI（核间中断）机制

arch/arm64/kernel/entry-common.cC 层异常处理分发
